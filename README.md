## Donkey

A language with a simple feature set following [Writing An Interpreter In Go](https://interpreterbook.com/)

## Progress

#### Lexer

Supported tokens

+ Operators: `=`, `+`
+ Delimiters: `,`, `;`, `(`, `)`, `{`, `}`
+ Identifiers:
  - type: integer, like 1, 2, 3
  - keywords: `let`, `fn`
  - other literals
+ EOF
+ Illegals
